# Redretep AI

A minimalist, web-based unblocked frontend for the Groq AI API.

This project features two visual modes: a classic Terminal/Matrix style for tech enthusiasts and a Notebook design for work.

* Multi-Model Support: Seamlessly switch between different LLMs (e.g., Llama 3.1 & 3.3).
* API Key Rotation: Automatically loads and rotates API keys from an external source to prevent rate-limiting issues.
* Intelligent History: Maintains context of the last 10 messages for fluid conversations.
* Regenerate Function: Re-run the last prompt with one click to get alternative responses.
* Modern Markdown: Full support for code highlighting via Prism.js and text formatting via Marked.js.
